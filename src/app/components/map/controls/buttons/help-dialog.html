MUSTANGular Help

MUSTANGular is an Angular application that displays station metrics from the IRIS MUSTANG service on a leaflet map. (Created by Pacific Northwest Seismic Network - PNSN - 2018)

See GitHub for more information.

General Information:

Data:

The metric information is from the IRIS MUSTANG metric service.
The station data is from the IRIS FDNSWS service.
The measurements information is from the IRIS MUSTANG measurements service.

Channel Value (across timespan)

MUSTANGular displays data for a station using one channel at a time. (See: see Display Value) The channel value is the aggregate of all of a channel's measurements. Possible values are:

  • maximum : highest measurement for the channel
  • minimum : lowest measurement for the channel
  • average : average of the measurements for the channel
  • median : median of the measurements for the channel
  • 5th_percentile : 5th percentile of the measurements for the channel
  • 95th_percentile : 95th percentile of measurements for the channel

Display Value

MUSTANGular displays data for a station using one channel at a time. (See: see Display Value) The channel value is the aggregate of all of a channel's measurements. Possible values are:

  • Channel Priority : shows a single channel, chosen by the user
  • Aggregate Channels : agregates all channel values to a single value

Channel Priority:

This display value option shows a single channel. The display channel is selected based on the user's prioritization of the channels. MUSTANGular will display data for a station using the first channel, in order of the prioritized list, that has data.

Channel Aggregate:

This display value option combines all of the channel aggregates into a single value. Options are:

  • maximum : highest measurement for the channel
  • minimum : lowest measurement for the channel
  • most extreme : maximum value of the absolute values

Binning:

The coloring of icons is determined by sorting all of the displayed values into bins. The upper and lower limits of binned values and the number of bins are selected by the user. The bins are even in width and inclusive at the lower and the upper boundary. Any values that are outside of the limits of the bins are categorized as high or low outliers.

Key:

The key has checkboxes for each "bin" that allows users to toggle the corresponding values on the map. The histogram on the key represents the proportion of stations that fall into that bin.

Station Markers:

Hovering over a dot on the map will show a list of channels available for that metric and the channel used for display (in green).

Station Chart:

Clicking on a station will show a graph of the metric data over the selected time range.

Copy Link:

Saves a link to the current metric view to the user's clipboard.

Download:

Opens the data in format of the selected file type in a new tab.
